## Code Book for Course Project Analysis Results

The following variables are included in the result set "Final":

1. Subject - Identifies the subject undertaking the experiment (1 through 30)
2. Activity - Character string describing the activity the subject performed and the other variables on the row relate to
3. Variables 3-35 describe the measures standard deviation averaged per activity & subject
4. Variables 36-68 describe the measures mean values averaged per activity & subject
